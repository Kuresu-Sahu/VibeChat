# VibeChat

Real-time, secure chat for communities & teams

## Overview

VibeChat is a lightweight full‑stack real‑time chat application built with a Vite + React frontend, a Node + Express backend, and Socket.IO for real‑time messaging. The project includes OTP-based authentication, media uploads (Cloudinary), and basic email integration. It is intended as a starting point for building community and team chat experiences.

## Tech stack

- Frontend: Vite, React
- Backend: Node.js, Express
- Real-time: Socket.IO
- Database: MongoDB (Mongoose)
- Media: Cloudinary
- Email: Nodemailer

## Key features

- Real-time messaging with Socket.IO
- OTP-based authentication for quick sign-in flows
- Image/file uploads (Cloudinary)
- User online presence tracking
- REST API for auth and message persistence
